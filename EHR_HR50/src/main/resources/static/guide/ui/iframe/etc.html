글자 색상

  • 폴더 위치에 있는 css 파일명은 테마명과 동일합니다. ex)그린테마의 경우, green.css
  • 테마에서 사용하는 이미지 폴더의 경우, /src/main/resources/static/common/ 폴더 아래에 테마명으로 폴더를 생성합니다.

테마확인

  • GNB 상단에 있는 테마설정에 있는 테마리스트를 확인합니다.
  • 2024.05.16 기준) Blue, Orange, Red, OrangeRed, GreenOrange, Gold, Green, Skyblue, Navy, White 총 10가지 테마를 제공합니다.
  • OrangeRed, GreenOrange, Gold의 경우, 투톤(두가지 색상) 그라데이션 테마입니다.

테마리스트 수정

  • 수정해야하는 공통 JSP 파일 : header.jsp
  • 수정해야하는 권한 별 JSP 파일 : main_ess_ceo.jsp(대표이사), main_ess_chef.jsp(조직장), main_ess.jsp(임직원 공통)
  • 수정해야하는 CSS 파일 : common.css, main_ess_leader.css, main_ess.css
