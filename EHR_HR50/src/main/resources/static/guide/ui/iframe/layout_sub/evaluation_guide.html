평가기초설정 (목록, 초기화면)

  • 공통 레이아웃 .sub_menu_container 에 클래스를 추가하여 각 페이지에 필요한 레이아웃 스타일을 추가함.

그 외 공통 wizard, table, tab과 sheet영역이 차지하는 레이아웃

  • 평가관리 페이지에서는 평가기초설정을 제외한 페이지에서 공통 wizard, table, tab을 사용하며 하단 영역은 sheet 영역이 차지한다.
  • 공통 wizard를 사용하므로 공통 레이아웃 중 wizard가 있는 경우의 레이아웃을 사용한다.
  • sheet영역은 디자인 상에서 height, width 값이 지정되어 있지 않은 경우 브라우저 하단과 상위 요소 width의 전체 영역을 차지하며 개발 시 ibsheet가 들어가는 영역이다.
  • 공통 요소와 시트 영역을 함께 쓰는 경우 section.sheet_section를 사용하고 section.sheet_section 내부 하단에 div.sheet 로 시트 영역을 표시한다.
