Layout

공통

  • 공통으로 body.iframe_content와 div.main_content로 되어 있으면 .main_content 내부에 해당 페이지에서 쓰이는 클래스를 가진 div요소를 추가하여 사용한다.
  • 컨텐츠 div 내부는 div.header와 div.widget_container로 구성되는데, .header에는 마스터페이지 타이틀과 오른쪽 상단의 버튼을 포함한다.
  • .main_content는 화면의 가운데 정렬 되어있으며 width값이 1116px;로 동일하다.

Widget

  • div.widget_container 내부에는 공통 widget과 동일한 구조와 사이즈의 위젯을 사용하며 배치도 메인의 위젯과 동일하다.
  • 위젯 내부는 공통 요소(dropdown, progress_bar, tab, chart 등)로 구성된다.

인사발령마스터

Layout

  • 인사발령마스터 페이지에서만 공통 widget 구성과 배치를 사용하지 않는다.
  • table 태그로 레이아웃을 분할하고 table.sheet_main 요소를 사용한다.
  • 레이아웃 내부에는 div.container 요소로 구성된다.
