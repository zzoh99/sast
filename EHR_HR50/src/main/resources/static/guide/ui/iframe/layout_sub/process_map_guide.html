Sortable

프로세스맵 에디터 Sortable

  • jquery의 sortable 함수를 이용하여 에디터 구현하여 필요시에 아래의 두 파일을 import하면 사용.
  • hrux-hr50/assets/js/process_map.js 파일에서 해당 에디터 영역의 id에 sortable 함수를 적용하여 사용.
  • div.box_wrap을 드래그 하여 이동하며, 드래그 도중에는 호버 시 보이는 상단 버튼을 숨겨놓았음.
  • 필요시 프로세스맵 개수에 따라 div#sortable 영역 안의 div.box_wrap의 수를 늘려서 사용.
  • div#sortable의 width 영역은 100%이며 가로의 .box_wrap의 수가 많아지거나 상위 요소의 width가 줄어들면 아래로 다음 줄로 내려감.
  • 프로세스맵 에디터의 sortable은 div.process_map_edit_container의 요소 내부에 있어 그에 따른 크기 영향을 받음.
