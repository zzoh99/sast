Input

Input

  • Input의 기본 클래스는 'form-input'이나, 상세페이지에서 사용하는 Input의 경우는 Table에 클래스가 종속적입니다.
  • Table 내부에서 사용하는 Input의 경우, Table&Form 페이지를 참고해주시기 바랍니다.

기본

<input class="form-input" type="text" />

ReadOnly

<input class="form-input" type="text" readonly />

Required

<input class="form-input required" type="text" />

Input 안에 있는 검색 버튼

search 
<div class="search_input">
    <input class="form-input" type="text" placeholder="사번/성명을 입력하세요">
    <i class="mdi-ico">search</i>
</div>

Checkbox

  • 체크박스의 경우, 심미성과 가시성을 위해 커스텀으로 스타일링 한 것을 사용합니다.
  • label과 함께 사용할 때에는 중앙정렬을 위해서 클래스 "form-wrap"을 감싸줍니다.
  • 웹접근성을 위해 label을 클릭시 체크 활성화를 지향합니다.
  • label 태그의 for 속성이 input 태그의 id값과 동일합니다.

기본

<input type="checkbox" class="form-checkbox type1"/>

기본(체크상태)

<input type="checkbox" class="form-checkbox type1" checked/>

disabled

<input type="checkbox" class="form-checkbox type1" disabled/>

disabled

<input type="checkbox" class="form-checkbox type1" disabled checked/>

label과 함께 쓰이는 checkbox

<div class="form-wrap">
    <input type="checkbox" class="form-checkbox type1" id="checkbox1" />
    <label for="checkbox1">선택</label>
</div>

label과 함께 쓰이는 checkbox 작은사이즈

<div class="form-wrap">
    <input type="checkbox" class="form-checkbox type2" id="checkbox2" />
    <label for="checkbox2">선택</label>
</div>

토글 스위치

<input type="checkbox" id="toggle" hidden />
    <label for="toggle" class="toggleSwitch">
    <span class="toggleButton"></span>
</label>
