레거시 테이블의 재사용

  • HR 5.0의 디자인은 HR 4.0의 마크업을 기준으로 hrux_fit.css 파일로 재스타일링을 했습니다.
  • table의 너비는 colgroup 내부에 선언합니다.

조회조건 테이블 .sheet_search table

  • 모든 페이지의 조회조건의 마크업에서 ".sheet_search table"의 클래스를 사용합니다.
