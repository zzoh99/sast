Icon

아이콘 (google font) 소개

  • HR 5.0에서 사용하는 아이콘은 모두 구글 Material Icon을 사용합니다.
  • Material Icon을 사용하면 크기 및 색상을 커스텀을 할 수 있습니다.
  • 필요한 아이콘의 경우는 https://fonts.google.com/icons에서 검색해 사용이 가능합니다.
  • LNB에서 사용하는 아이콘 역시 mdi 아이콘을 사용합니다.

아이콘 사용하는 방법

assignment_ind 
<i class="mdi-ico">assignment_ind(아이콘 명)</i>

스타일 클래스

  • material Icon Outline 을 사용할 때에는 'mdi-ico'를 기본 클래스로 사용합니다.
  • material Icon 에서 채워진 형태의 아이콘을 사용할 때에는 'mdi-ico filled' 나 'mdi-ico round'를 사용합니다.

기본

settings 
<i class="mdi-ico">settings</i>

filled

settings 
<i class="mdi-ico filled">settings</i>

round

settings 
<i class="mdi-ico round">settings</i>
