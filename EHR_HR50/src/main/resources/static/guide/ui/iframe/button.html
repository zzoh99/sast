Button

Basic

  • 버튼의 스타일의 종류는 색상/크기/아이콘의 여부로 나눠집니다.
  • 아이콘이 들어가는 버튼의 경우, 구글폰트아이콘(https://fonts.google.com/icons)을 참고하거나, 가이드 문서 내 아이콘 페이지를 참고해주시기 바랍니다.
  • 버튼의 기본 클래스는 'btn' 입니다.

외곽선 있는 버튼1

  • 테마에 관계없이 일정한 스타일로 일반 버튼에서 주로 쓰입니다.
<button class="btn outline_gray">Outline_gray</button>

외곽선 있는 버튼2

  • 테마에 관계없이 일정한 스타일로, 강조하고 싶은 버튼에 주로 사용합니다.
<button class="btn outline">outline</button>

색상이 있는 기본 버튼

  • 테마 색상에 따라 버튼의 색상이 바뀝니다.
<button class="btn filled">Filled</button>

텍스트 형태의 버튼

  • hover 시 버튼 모양처럼 opacity가 보여지는 버튼
<button class="btn Text">Text</button>

아이콘과 함께쓰이는 버튼

  • 아이콘 사용방법은 'Icon' 페이지 참고
<button class="btn outline_gray icon_text">
    <i class="mdi-ico filled">settings</i>설정
</button>

아이콘 단독으로 버튼

  • 상단 GNB 영역에 있는 버튼
notifications 
<a href="#" class="user-alert">
  <i class="mdi-ico filled" title="알림">notifications</i>
</a>

버튼 크기

  • 버튼 기본 크기는 border를 포함하여 33px 입니다.
  • 버튼 크기는 3가지 종류로 기본/thin/thinner로 나뉩니다.

버튼 크기 thin

  • thin 버튼 크기는 border를 포함하여 28px 입니다.
<button class="btn outline_gray thin">thin</button>

버튼 크기 thinner

  • thinner 버튼 크기는 border를 포함하여 24px 입니다.
<button class="btn outline_gray thinner">thinner</button>
